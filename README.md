# devcontainer-features

Collection of `.devcontainer` features for Android reverse engineering. A fork of [NordcomInc/devcontainer-features](https://github.com/NordcomInc/devcontainer-features).

## Android

### `ghcr.io/zolbooo/android-devcontainer-features/android-sdk:1`

Setup and update the Android SDK. [View source](https://github.com/zolbooo/android-devcontainer-features/tree/main/src/android-sdk).

### `ghcr.io/zolbooo/android-devcontainer-features/apktool:1`

Install Apktool CLI from official GitHub releases. [View source](https://github.com/zolbooo/android-devcontainer-features/tree/main/src/apktool).

### `ghcr.io/zolbooo/android-devcontainer-features/frida:1`

Install Frida CLI/tools and Android Frida server or gadget artifacts. [View source](https://github.com/zolbooo/android-devcontainer-features/tree/main/src/frida).

### `ghcr.io/zolbooo/android-devcontainer-features/jadx:1`

Install JADX CLI and GUI from official GitHub releases. [View source](https://github.com/zolbooo/android-devcontainer-features/tree/main/src/jadx).

### `ghcr.io/zolbooo/android-devcontainer-features/radare2:1`

Install Radare2 CLI from official GitHub releases. [View source](https://github.com/zolbooo/android-devcontainer-features/tree/main/src/radare2).
